Developer Experience

API usage comparison between Orca and Walrus

Orca

// 1. Upload video const res = await fetch('/api/upload', { method: 'POST', headers: { 'X-API-Key': key }, body: formData }); const { id } = await res.json(); // 2. Get HLS manifest URL const manifest = `/stream/${id}/index.m3u8`; // 3. Play with hls.js const hls = new Hls(); hls.loadSource(manifest);
  • Simple REST API
  • HLS ready out of box
  • Low latency streaming
  • Centralized dependency

Walrus

// 1. Upload to Walrus Publisher const res = await fetch('Publisher API', { method: 'POST', body: videoBlob }); const { blobId } = await res.json(); // 2. Fetch from Aggregator const manifest = `Aggregator/${blobId}/index.m3u8`; // 3. Need custom HLS handling // + blob mapping management
  • Truly decentralized
  • No single point of failure
  • Global availability
  • Higher latency
  • Complex integration
  • No native HLS support
